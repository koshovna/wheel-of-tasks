# Wheel of Fortune — Спецификация

## Концепция

Минималистичное колесо для выбора случайной задачи из списка дел. Мягкий, спокойный дизайн без азартных элементов казино.

## Дизайн

- **Эстетика:** Мягкий, светлый, минималистичный
- **Шрифт:** Nunito (Google Fonts)
- **Цвета:** Светлый фон (#f8f9fc → #eef1f8), индиго акцент (#6366f1)
- **Формы:** Округлые карточки, мягкие тени

## Архитектура

### Backend (Java + Jetty)
- `WheelServer.java` — HTTP сервер на Jetty
- REST API эндпоинты:
  - `GET /api/tasks` — список задач
  - `POST /api/tasks/add` — добавить задачу
  - `POST /api/tasks/delete` — удалить задачу
  - `GET /api/spin` — вращение (возвращает угол + победителя)
- Данные хранятся в `tasks_data.json`

### Frontend (HTML + JS)
- Canvas для отрисовки колеса
- API вызовы для CRUD операций
- Анимация вращения на клиенте

## Функционал

- ✅ Добавление задач
- ✅ Удаление задач (минимум 2)
- ✅ Регулируемое время вращения (2-10 сек)
- ✅ Сохранение задач на сервере
- ✅ Корректное определение победителя (бэкенд)
- ✅ Подсветка выбранной задачи
- ✅ Модальное окно с результатом
- ✅ Переключение светлой/тёмной темы (с сохранением в localStorage)
